## 🚗 Project Overview

A real-time Electric Vehicle Dashboard combined with an Advanced Driver 
Assistance System (ADAS) warning system, built during the Emertxe 
embedded systems internship program.

---

## ⚙️ Features

- Real-time vehicle speed, battery level, and temperature display
- ADAS proximity-based warning system with alert triggers
- Pure Embedded C implementation
- Hardware-software co-design on embedded target
